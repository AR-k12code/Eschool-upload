<p>Home Access Center (HAC) is available for you to access {{Student_name}} school information over the Internet.</p>
<p>Through HAC, you can view a daily summary of your student's attendance, schedule, and classwork.  In addition, interim progress reports, report cards, and discipline information may be available.</p>
<p>To access HAC, use the following URL: https://hac20.esp.k12.ar.us/HomeAccess20 or use the link on our website at https://www.gentrypioneers.com</p>
<p>An Access Code has been created for you to access your child's HAC record.  Follow the steps below to use your access code...</p>
<p>1.  Access the HAC Website at https://hac20.esp.k12.ar.us/HomeAccess20</p>
<p>2.  Below the Password box click on the link labeled <i>Click Here to Register With Access Code.</i></p>
<p>3.  Select the district name Gentry School District</p>
<p>4.  Enter your Access Code, <strong>{{Guardian_accesscode}}</strong>  in the Access Code box.</p>
<p>5.  Enter the birthdate of one of your children in a MM/DD/YYYY format.</p>
<p>6.  Select Sign In and the focus moves to the My Account page.</p>
<p>7.  Enter the desired password in the New Password fields.</p>
<p>8.  You must compose and answer three,  security challenge questions  if you have not done so before.</p>
<p>Any questions and answers entered here should be recorded and kept in a secure location.</p>
<p>This information can be used to access HAC in the case of a forgotten or an expired password.</p>
<p>9.  Once the password is entered and the required security question(s) are created, select the button labeled <i>Continue to Home Access Center</i>, which opens the HAC login screen.</p>
<p>At this point, select Gentry School District and enter your login and password to access HAC.</p>
